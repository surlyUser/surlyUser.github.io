
# Project SSG

This is a simple Website using Github Pages and SSG created for a College project.

## Authors

- [@surlyUser.github.io](https://www.github.com/surlyUser)


## Documentation

This site builds upon the standard theme https://pages-themes.github.io/architect/.

All one needs are:
- md-files
- additional html- and scss-files to customize the theme

Depending on ones needs the use of _includes and liquid-templates might be neccessary.
